### **Java Development Tools Setup & Installation**  

#### **1. JDK (Java Development Kit) & JVM (Java Virtual Machine)**  
- Required to **compile and run** Java programs.  
- **Types of JDK:**  
  - **Oracle JDK** (Licensed, paid support, used by enterprises)  
  - **OpenJDK** (Free, open-source alternative)  
  - **Amazon Corretto** (Amazon’s distribution of OpenJDK)  
- **Latest Version:** JDK 22 (Java updates every 6 months)  
- **LTS Versions:** Java 8, 11, 17 (Long-Term Support)  
- **For learning, use:** Java 20  

#### **2. JDK Installation Steps**  
1. Choose JDK based on your OS (**Windows, Linux, macOS**).  
2. Download the JDK from the OpenJDK website.  
3. Install and set up environment variables if required.  

---

### **3. IDEs (Integrated Development Environments)**  
- **Purpose:** Provides an editor, compiler, debugger, and tools in one place.  
- **Popular IDEs for Java:**  
  1. **Eclipse** – Open-source, widely used.  
  2. **VS Code** – Lightweight, supports multiple languages.  
  3. **IntelliJ IDEA** – Advanced features, popular for professional development.  

#### **4. IDE Installation Steps**  
##### **Eclipse**  
1. Go to the official **Eclipse** website.  
2. Choose **Eclipse Java EE (Enterprise Edition)** for full features.  
3. Download the installer for your OS and install it.  

##### **VS Code**  
1. Search **"Download VS Code"** or visit:  
   **[https://code.visualstudio.com/download](https://code.visualstudio.com/download)**  
2. Select the right version for your OS.  
3. Install the downloaded file.  

##### **IntelliJ IDEA**  
1. Visit the **IntelliJ IDEA** website.  
2. Click **Download** and choose your OS.  
3. Select **Community Edition** (Free version).  
4. Install and set up accordingly.
